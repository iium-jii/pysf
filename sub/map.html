오시는 길

행사주소

주소. 충남 부여군 규암면 백제문로 400 (우)33115

전화. 041-939-1000

길찾기

자동차로 오시는 경우

  • 서울(경부고속도로)
    서울 → 경부고속도로 → 천안JC → 천안논산고속도로 → 서공주JC → 공주서천고속도로 → 부여IC → 롯데리조트부여 (약 2시간 30분 소요)
  • 서울(서해안고속도로)
    서울 → 서해안고속도로 → 당진JC → 당진고속도로 → 서공주JC → 공주서천고속도로 → 부여IC → 롯데리조트부여 (약 2시간 소요)
※ 소요 시간은 교통 상황에 따라 다를 수 있습니다.

버스로 오시는 경우

  • 서울 출발
    남부 터미널(02-551-8550) → 부여 시외버스 터미널 (약 2시간 소요)
    동서울 터미널(02-446-8000) → 부여 시외버스 터미널 (약 2시간 40분 소요)
  • 천안 출발
    천안 시외버스 터미널(041-640-6400) → 부여 시외버스 터미널 (약 2시간 소요)
  • 대전 출발
    서부 시외버스 터미널(042-584-1615) → 부여 시외버스 터미널 (약 1시간 30분 소요)
  • 청주 출발
    서부 시외버스 터미널(1688-4321) → 부여 시외버스 터미널 (약 3시간 10분 소요)
  • 농어촌버스
    부여 시외버스 터미널 : 403번(부여시장) → 한국전통문화대학교 (약 20분 소요)
    * 농어촌 버스 시간표문의 : 041-843-6092~3
※ 예약 또는 기타 자세한 사항은 전화, 또는 홈페이지를 통해 확인하시기 바랍니다.

기차로 오시는 경우

  • 서울 출발
    용산역 → 논산역 (약 1시간 34분 소요)
    용산역 → 공주역 (약 59분 소요)
  • 대구 출발
    대구역, 동대구역 → 대전역 (약 40분 소요)
  • 부산 출발
    부산역 → 대전역 (약 2시간 소요)
  • 광주 출발
    광주송정역 → 논산역 (약 53분 소요)
    광주송정역 → 공주역 (약 45분 소요)
※ 소요시간 기준은 가장 빠른 KTX 기준입니다.
※ 하차역에서 부여 시외버스 터미널로 환승하신 후, 대중교통을 이용해 롯데리조트부여로 오시면 됩니다.
※ 예약 또는 기타 자세한 사항은 우측 상단의 기차 예매 버튼, 코레일 홈페이지, 전화로 문의하시기 바랍니다.
